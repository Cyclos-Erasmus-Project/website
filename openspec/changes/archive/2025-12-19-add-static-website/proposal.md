# Change: Create CYCLOS Static Website

## Why

The CYCLOS ERASMUS+ project requires a public-facing website to communicate its mission, showcase partner organizations, share project results, and provide access to educational resources. This is essential for WP5 (Dissemination) and supports the project's visibility and impact across the 4 partner countries.

## What Changes

- Initialize Astro project with TypeScript and Tailwind CSS
- Create responsive, accessible homepage with project overview
- Build partner organizations showcase page
- Implement multilingual support (EN, FR, RO, DE, EL)
- Create about/objectives page explaining project goals
- Add news/updates section for project progress
- Include EU ERASMUS+ funding acknowledgment and disclaimer
- Set up contact page with partner contact information
- Implement mobile-first responsive design
- Ensure WCAG 2.1 AA accessibility compliance

## Impact

- **Affected specs**: `website` (new capability)
- **Affected code**: Entire new codebase
- **New dependencies**: Astro, Tailwind CSS, astro-i18n
- **Partner visibility**: All 4 organizations showcased
- **Compliance**: EU funding requirements met
